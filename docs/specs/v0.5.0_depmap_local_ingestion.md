# v0.5.0 DepMap local ingestion

Issue 502 extends the Issue 501 canonical `targetintel.functional_dependency`
contracts. The existing release manifest, schema fingerprint, local-layout
contract, and `validate_local_release` validator remain the sole release and
checksum architecture. `depmap_ingestion.py` is the sole matrix ingestion
surface; it has no network client or downloader.

`target_subset` requires an explicit target universe. Its canonical,
order-independent identity is part of the request and snapshot identity. It
inspects every matrix header, retains exact resolved columns only, and records
unresolved, ambiguous, invalid, and duplicate requests. `full_matrix` rejects a
target universe, validates each declared matrix sequentially, writes indexes,
and references the source files through the validated manifest rather than
copying complete raw matrices.

Before parsing, every declared file is checked against the manifest for safe
location under the explicit data root, presence, byte size, SHA-256, supported
format, and header schema. Validation failures are terminal and no valid
snapshot is written. Optional roles remain explicitly unavailable when absent.

The versioned `depmap-gene-label-v2` parser accepts only exact
`SYMBOL (ENTREZ)` labels, including source-preserved mixed-case approved symbols
such as `C9orf72 (79087)`. It preserves the original label, parsed fields,
status, and limitation. There is no alias, case-insensitive, fuzzy, or
punctuation repair. Duplicate canonical identities remain separate source
columns. Matrix values are strictly numeric; blank values remain blank and
invalid values fail parsing. Model joins use only `ModelID`;
duplicate or blank IDs fail, while unmatched matrix and metadata models are
retained in reconciliation indexes.

Artifacts use stable names and canonical JSON: `ingestion_snapshot.json`,
`coverage_summary.json`, `gene_index.tsv`, and `model_index.tsv`; subset mode
also writes the two subset TSV files. Scientific identity excludes roots,
output locations, timestamps, actor, host, and temporary paths, while including
release checksums/schemas, parser and mapping versions, mode, and target
universe where applicable. Paths are explicit, traversal is rejected, outputs
cannot be beneath source files, and writes are atomic where practical.

The fixture is synthetic and non-biological. Production matrices are supplied
locally by the caller and processed sequentially so the two wide matrices are
not retained simultaneously by design. These outputs are the handoff for Issue
503; this issue calculates no dependency profile, melanoma universe, essential
context, selectivity, score, rank, role, or clinical conclusion.
